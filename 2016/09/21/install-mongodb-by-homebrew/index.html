install mongodb by homebrew

首先，安装homebrew

详见官网。哈。

然后是

  1. brew update 更新brew
  2. brew install mongodb 安装mongodb数据库
  3. brew services start mongodb 启动mongodb数据库
  4. brew services stop mongodb 停止mongodb数据库

安装完会有提示
